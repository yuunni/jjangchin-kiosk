# 개요
구매 내역 분석을 통한 상품 추천 키오스크를 개발한다. 이를 통하여 사용자들이 키오스크를 사용하는 시간이 줄어들어 키오스크 앞의 인구 밀집도를 낮출 수 있다. 또한 사용자 기다림을 줄여 점진적으로 키오스크에 투자되는 비용을 감소시키며 감염 확산을 방지한다. 또한 키오스크의 UI/UX를 개선하여 사용에 어려움을 겪는 사람들에게 더욱 간편한 결제를 제공한다.
